Этап установки фреймворка yii2:

Установка yii2:

  • Скачивание архива с официального сайта yii2
  • Загрузка содержимого из архива в OpenServer
  • Прописывание ключа валидации

    На данном этапе разработки был установлен фреймворк Yii2, который в дальнейшем сыграет очень важную роль в разработке CRUD-таблицы

    Разработка и проектирование базы данных для CRUD-таблицы:

    Этапы разработки бд:

  • Проектирование БД
  • Составление примерного макета БД
  • Создание БД

    В ходе данного этапа разработки была спроектирована и реализована база данных для записи данных в неё информации (хранение), и для вывода данных в таблицу (обработка)

    Разработка самой CRUD-таблицы и её функционала

    Этапы разработки CRUD-таблицы:

  • Создание модели посредству Gii
  • Создание CRUD-таблицы посредству Gii
  • Установка пути до таблицы

    В ходе данного эпапа была реализована посредству Gii CRUD-таблица, в которой:

  • Была реализована функция вывода информации из базы данных
  • Была реализована функция создание "поста"
  • Была реализована функция удаления "поста"
  • Была реализована функция изменения информации в "посте"
  • Была реализована функция поиска информации по таблице

    Создание и отправление данных в репозиторий на GitHub

    Этапы создание и отправления данных в репозиторый:

  • Создание нового репозитория
  • Добавление комментариев к файлам в репозиторие
  • Отправление данных в репозиторий

    Таким образом были сохранены все файлы небходимые для работы с CRUD-таблицой на GitHub
